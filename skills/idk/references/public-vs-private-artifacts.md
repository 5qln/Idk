# Public vs. Private Artifacts — The Membrane at X|α

The membrane rule: **only X (the validated question) crosses the membrane.**
Everything from G-phase onward (α, {α'}, Z, ∇, B'', ∞0') is private.

## The Membrane

```
S:  ∞0 → ?          [private]
    ↓
X:  QUESTION         [PUBLIC — crosses membrane |]
    ↓
G:  α ≡ {α'}        [private]
    ↓
Q:  φ ⋂ Ω → Z       [private]
    ↓
P:  δE/δV → ∇       [private]
    ↓

    ↓
∞0' → new X          [PUBLIC — crosses membrane |]
```

## Two Artifacts Per Cycle

| Artifact | Visibility | Format | Purpose |
|----------|-----------|--------|---------|
| **Public question (X)** | Published to commons | Single markdown file: question text + CC0 + hash | Propagation. Someone else's starting point. |
| **Private trail** | Local only | Full cycle format (see `cycle-trail-format.md`) | Certificate of authentic inquiry. Matching in the dark. |

## Public Question Format

```markdown
# [Question text]

*Published: YYYY-MM-DD. CC0-1.0. Hash: <sha256>*
```

Nothing else. No α. No Z. No ∇. No identity. Just the question.

## Private Trail Format

The full cycle trail (X, α, Z, ∇, B'', ∞0') is the private certificate.
It proves authentic inquiry occurred — all gates passed, the chain is intact.
It enables signature-to-signature matching (structural coherence detected
across private trails without revealing content). It is never published.

## Propagation Chain

```
X (public) → private trail → ∞0' → new X (public)
```

The trail is invisible infrastructure connecting visible questions.
One person's published X becomes another person's starting point.
The trail connecting them is private. Only the questions are public.

# Trail-Commons — Agent-Native Architecture

> Status: scaffold. Cited from `SKILL.md`. Authoritative design lives in the
> `Idk-commons` repository (`ARCHITECTURE.md`); summarise it here.

Agent-native architecture for moving questions across the privacy membrane:

- **Git as transport** — no server, no accounts, no tokens; questions move as
  commits/bundles.
- **Trail format as protocol** — the private trail is structured, but only the
  published question (X) and the return question (∞0′) are allowed to cross.
- **/idk publish · discover · browse** — the agent verbs over the commons.

Reference this when ∇ points to publishing or sharing trails beyond a single
agent session. See `Idk-commons/skills/trail-commons/` for the implementation.
